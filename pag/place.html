MEDICAL CENTER



Cada una de las sedes cuentan con una estructura edilicia que facilita el desarrollo de las investigaciones. Destacando:

    Laboratorio de análisis clínicos y microbiológicos apto para el procesamiento y almacenamiento de muestras, con Control de Calidad Externo certificado (según norma ISO 9001:2015).

    Área de Diagnóstico por imágenes con tomógrafo y resonador de última generación.

    Farmacia con espacio para el almacenamiento de drogas en investigación, con personal destinado para realizar los controles de temperatura correspondientes.

    Consultorios externos para la atención ambulatoria de pacientes y la asignación de áreas diferenciada para la atención de pacientes pediátricos y oncológicos.

    Mobiliario con seguridad y heladeras para el almacenamiento de medicación, con termómetros para el registro de temperatura actual, mínima y máxima.

    Hospital de día con vacante para infusión y personal de enfermería entrenado en Buenas Prácticas Clínicas.

    Oficinas con acceso restringido destinadas para el archivo de documentación confidencial.

    Oficinas para coordinadores de estudio.

    Salas de monitoreo con conexión a Internet y aire acondicionado.
